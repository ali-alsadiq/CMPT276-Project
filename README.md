## PROJECT ABSTRACT

Ascend is a web-based fitness tracking application that focuses on keeping users consistent rather than overwhelming them with features. Many current fitness apps require constant manual input, contain unnecessary tools, or place useful functionality behind subscriptions. As a result, users often stop using them even when they want to improve their habits. Ascend is designed to reduce that friction by limiting the interface to essential tracking tools and providing motivation through built-in progression rewards.

The problem at hand attempting to be solved by current fitness applications (like myfitnesspal, calAI, etc.) is that they are cluttered with features that retract from the users requirements. These applications rely entirely on self discipline to manually enter information without a meaningful incentive for the user, thus making them less likely to stay on track. Users feel overwhelmed by sub-features, and under-stimulated from the lack of reward. Additionally, many applications lock their most desired features behind a subscription-based paywall, making effective fitness habit-forming inaccessible.

Our application is focused on simplicity and focused user requirements that reward consistency. By integrating robust APIs (FatSecret & API-Ninjas Exercise), (app name) makes it easy for anyone to track their goals. Data is then used to create a positive reinforcement loop, rewarding users in a gamified ranking system. This makes life better by making macro/exercise tracking frictionless and offering immediate reward for the users progression. We aim to blend education and entertainment through teaching users about macro-nutrient sufficiency and a fun, progression-based ranking system.

Ascend targets individuals who want accountability but feel discouraged by complicated fitness platforms. The application aims to make tracking straightforward while adding light gamification to keep users engaged. By combining clear feedback with achievable progression, Ascend functions both as a practical habit-building tool and an engaging daily activity.


### Epic 1 (Insights & Progress Page): 
Main dashboard section that displays a weekly and/or daily tracker indicating whether the user’s goals were reached. This page will dynamically calculate a consistency score based on goal completion for a ranking system that gamifies the fitness process. Additionally information such as a goal streak, weekly/monthly fitness trends, feedback, and recommendations are all shown on this page.

### Epic 2 (Calorie / Nutrition Tracker): 
This page will include a calculated calorie goal, and display specific sections for macronutrient tracking such as, protein, carbs, and fats. A key feature of this page will be a food logging section where users are able to search for foods, and modify their quantities to get a breakdown of its nutrients.
https://platform.fatsecret.com/platform-api

### Epic 3 (Health & Wellness Tracker): 
This page allows the user to log exercise sessions and calculate total calories burned via the type of exercise and duration of the exercise. API allows options such as exercise type, and exercise intensity. Tracks how many workout days were completed each week, and integrates into the ranking system.
https://api-ninjas.com/api/caloriesburned 

### Epic 4 (User Section): 
Handles account creation, login/logout, security. In this page, users can edit profile details (display name, username, bio, birthday, etc). Personal preferences and settings are able to be accessed after initially chosen in this age.
